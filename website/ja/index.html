退屈もしなければ、あなたを値踏みもしない AI と話して、言語を学ぶ。— 望むなら — インターネットには一度もつながずに。

流れを止めないその場の添削つきの会話、AI が生成するレベル別リーディング、語彙の自動記録、そして話す側にも聞く側にも声。すべてはあなたのディスク上の SQLite ファイルの中にあります。登録すべきサーバーはありません。そもそもサーバーが存在しないからです。

セッションはメニューではなく、計画です。 ひとつのテーマが、今日の会話・リーディング・ロールプレイ・復習期日の来た単語を貫きます。何を勉強しますかとは訊かれません。↵ を押せば、始まります。

セッション

コーチが先に口を開き、会話のテンポを保ちます。

添削は会話の流れの中にそのまま現れ、つまずいた単語は静かに書き留められます。ヒントは詰まったときのためにあります。あなたの代わりに答えるためではありません。

音声も口述入力も、設定なしで。

21 MB の音声がアプリの中で動きます。キーもサーバーも要らず、ネットワークには何も出ていきません。話す、文字になる、声で返ってくる。そして Verba は、いまどの段階が実際に話しているのかを必ず教えてくれます。

Verba を選ぶ理由

6 つの決断と、その代償。

オフライン、本当の意味で。

Ollama をつなげば、チャットもリーディングも音声も口述入力も、アプリ全体が飛行機の中で動きます。アカウントは不要、ずっと。

プロバイダーもキーも、あなたのもの。

Ollama、LM Studio、OpenAI、Anthropic、Gemini、OpenRouter。設定画面で差し替えられます。囲い込みは一切ありません。

声が返ってきます。

話せば文字になり、答えは声で返ってきます。同梱モデルは必要になったときにダウンロードされ、アプリ内でそのまま動きます。

覚えています。

つまずいた単語は間隔反復のデッキになり、そのデッキが明日のセッションを組み立てます。

データは、そこから動きません。

すべてはローカルの SQLite ファイルの中にあります。アップロードもしません、同期もしません、売りもしません。

8 言語、追加はフォルダひとつ。

英語、スペイン語、フランス語、ドイツ語、日本語、トルコ語、イタリア語、ポルトガル語。あなたの言語を追加。

音声

はしごを上から順に。最初に応えた段が、声になります。

Verba は 同梱 → ローカルサーバー → クラウドのキー → OS の音声 の順に試し、いまどの段階が実際に話しているのかを必ず教えてくれます。既定は同梱の段階で、これはあなたに何も要求しません。

モデル役割サイズ速度
Piperデフォルト 音声 — 言語ごとに 1 つ ~21 MB ~7× 実時間
Kokoro 音声 — 53 の声、6 言語 349 MB ~2× 実時間
Whisper 口述入力 — base / small 208 / 639 MB 実時間より速い

Piper のダウンロードサイズは Kokoro の 50 分の 1、速度は数倍です。会話の一往復にとっては、自然さの最後の数パーセントよりもそちらのほうがずっと効きます。すでに自前の音声サーバーをお使いですか。OpenAI 互換のものならどれでも動きます（Kokoro-FastAPI、speaches、LocalAI、vLLM）。ElevenLabs や Deepgram のキーも使えます。

Mac では、Apple Silicon なら aarch64 を、Intel Mac のときにかぎり x64 を選んでください。取り違えると、起動を拒まれるか、Rosetta の上を這うことになります。

このビルドは署名されていません。それでも開く方法はこちらです

Verba には Apple Developer 証明書も、Windows のコード署名証明書もありません。どちらかの後ろにビルドを閉じ込めておくくらいなら、そのまま出すほうを選びました。そのため、どちらの OS も最初の一度は起動を拒みます。

macOS ではこう出ます。"「Verba」にマルウェアが含まれていないことを Apple は検証できませんでした" — ゴミ箱に入れる をクリックし、アプリを /Applications にドラッグしてから xattr -d com.apple.quarantine /Applications/Verba.app を実行してください。 

xattr -dr com.apple.quarantine /Applications/Verba.app

ターミナルを使わないなら、一度開こうとしたあとで システム設定 → プライバシーとセキュリティ → 下までスクロール → このまま開く と進みます。

Windows ではこう出ます。"PC が保護されました" — 詳細情報 をクリックしてから 実行 を選んでください。

自分でビルドすればどちらも回避できますし、ここまでの手順に抵抗があるなら、それが誠実な選択肢です。これは Verba にかぎった話ではありません。署名のないオープンソースのデスクトップアプリが、みな等しく払っている通行料です。

プラットフォームについて、正直に。 4 つのターゲットはすべて同じソースから CI がパッケージしていますが、同梱の音声エンジンが実際に動いた実績があるのは macOS arm64 だけです。Windows や Linux で最初に動かしたのがあなたで、声が出てこなかったなら、それは報告する価値のあるバグです。あなたのやり方が間違っているわけではありません。

あるいは、ソースからビルドする

git clone https://github.com/nuvocode/verba.git
cd verba
npm install
npm run tauri build

Node と Rust が必要です。オフラインで使うなら、Ollama を起動しておいてください。
