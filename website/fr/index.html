Une séance est un plan, pas un menu. Un même thème relie la conversation, la lecture, le jeu de rôle et les mots à réviser aujourd'hui. On ne vous demande jamais quoi étudier — vous appuyez sur ↵ et vous commencez.

La séance

Le coach parle en premier et garde l'échange en mouvement.

Les corrections arrivent en ligne, dans le fil de la conversation, et le mot sur lequel vous avez buté est discrètement noté. Les suggestions sont là pour le moment où vous bloquez — pas pour répondre à votre place.

Voix et dictée, sans rien à configurer.

Une voix de 21 MB tourne à l'intérieur de l'application : pas de clé, pas de serveur, rien sur le réseau. Parlez, faites-vous transcrire, recevez une réponse à voix haute — et Verba vous dit toujours quel niveau parle réellement.

Voix

Une échelle, gravie dans l'ordre, à voix haute.

Verba parcourt modèle fourni → serveur local → clé cloud → voix du système et vous dit toujours quel niveau parle réellement. Le niveau fourni est celui par défaut, et c'est celui qui n'attend rien de vous.

ModèleRôleTailleVitesse
Piperpar défaut Voix — une par langue ~21 MB ~7× le temps réel
Kokoro Voix — 53 voix, six langues 349 MB ~2× le temps réel
Whisper Dictée — base / small 208 / 639 MB plus rapide que le temps réel

Piper pèse un cinquantième du téléchargement de Kokoro et va plusieurs fois plus vite — ce qui, pour un tour de parole, compte davantage que les derniers pour-cent de naturel. Vous faites déjà tourner votre propre serveur vocal ? N'importe lequel compatible OpenAI fonctionne (Kokoro-FastAPI, speaches, LocalAI, vLLM), tout comme les clés ElevenLabs et Deepgram.

Sur un Mac, prenez aarch64 pour Apple Silicon et x64 uniquement pour un Mac Intel — le mauvais refuse de démarrer ou rampe sous Rosetta.

La version n'est pas signée — voici comment l'ouvrir quand même

Verba n'a ni certificat Apple Developer ni certificat de signature de code Windows, et nous préférons publier la version plutôt que de la bloquer derrière l'un ou l'autre. Les deux systèmes d'exploitation refuseront donc de l'ouvrir au premier essai.

Sur macOS, vous verrez : "Apple n'a pas pu vérifier que « Verba » ne contient pas de logiciels malveillants" — cliquez sur Déplacer vers la corbeille, puis glissez l'application dans /Applications et lancez xattr -d com.apple.quarantine /Applications/Verba.app. 

xattr -dr com.apple.quarantine /Applications/Verba.app

Sans terminal : essayez de l'ouvrir une fois, puis Réglages Système → Confidentialité et sécurité → faites défiler vers le bas → Ouvrir quand même.

Sur Windows, vous verrez : "Windows a protégé votre ordinateur" — cliquez sur Informations complémentaires puis sur Exécuter quand même.

La compiler vous-même contourne les deux, et c'est l'option honnête si ce qui précède vous met mal à l'aise. Rien de tout cela n'est propre à Verba — c'est le péage que paie toute application de bureau open source non signée.

Honnêteté sur les plateformes. Les quatre cibles sont empaquetées depuis la même source par la CI, mais le moteur vocal fourni n'a jamais tourné ailleurs que sur macOS arm64. Si vous êtes le premier à le lancer sur Windows ou Linux et qu'aucune voix n'en sort, c'est un bug qui mérite d'être signalé — pas vous qui vous y prenez mal.

Ou compilez-la depuis les sources

git clone https://github.com/nuvocode/verba.git
cd verba
npm install
npm run tauri build

Nécessite Node et Rust ; pour l'offline, Ollama en marche.
