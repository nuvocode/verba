# Portuguese (Português)

**Variety:** Brazilian Portuguese — `você` as the everyday register, Brazilian
pronunciation. European Portuguese differs enough that it deserves its own pack
rather than a footnote here.

**Maintained by:** the community contributor who submitted it, per
[CONTRIBUTING.md](../../../../../CONTRIBUTING.md) (Community pack — reviewed by one
maintainer and one fluent speaker).

Only `pack.ts` is required; this folder has no long-form docs yet. See
[`../es/`](../es/) for the reference layout if you want to add some.
