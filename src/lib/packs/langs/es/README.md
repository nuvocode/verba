# Spanish (Español)

The reference language folder. Every other language may copy this layout;
nothing here except `pack.ts` is required.

| File | What it is |
| --- | --- |
| `pack.ts` | The pack literal — the app's contract. Short, structured, always loaded. |
| [`pronunciation.md`](./pronunciation.md) | Long-form sound notes for the learner. |
| [`grammar.md`](./grammar.md) | Grammar the three `grammar` bullets have no room for. |
| [`register.md`](./register.md) | `prompt: true` — also fed to the tutor on every turn. |

Docs are shown in **Settings → Language**, index (`README.md`) first, then by
filename. Prefix with `01-`, `02-` if a language needs a reading order.

## Scope of this pack

Neutral Spanish: peninsular spelling and vocabulary, avoiding regionalisms that
would confuse a Latin-American learner (and vice-versa). `vosotros` is
recognised but never taught first — `ustedes` covers both audiences.
