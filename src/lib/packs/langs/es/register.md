# Register and courtesy

Address the learner as `tú` in everyday scenarios (friends, cafés, shops, small
talk). Switch to `usted` — and stay there — when the scenario is formal: a job
interview, a doctor, a police officer, a hotel front desk, an older stranger.
If the learner uses the wrong one for the situation, correct it as a **minor**
issue and say which register the scene calls for.

Prefer `ustedes` for plural "you" so the reply works for both Spain and Latin
America. Use `vosotros` only if the scenario is explicitly set in Spain.

Spanish softens requests with the conditional and with `poder`, not with please:
`¿Me podrías ayudar?` beats a bare imperative plus `por favor`. Note this when
the learner's Spanish is grammatical but reads as brusque.
