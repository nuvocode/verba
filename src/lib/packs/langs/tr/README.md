# Turkish (Türkçe)

**Variety:** standard Istanbul Turkish, as written and spoken in Turkey.

**Maintained by:** core team (Official pack).

Only `pack.ts` is required; this folder has no long-form docs yet. Vowel harmony
and the suffix chain are the obvious first chapters — see [`../es/`](../es/) for
the reference layout and [CONTRIBUTING.md](../../../../../CONTRIBUTING.md) for
the rules.

## Speech

Turkish is the language that shows why `recommendedVoices` exists. Kokoro — the
bundled tier's large multilingual voice — has **no Turkish speaker at all**; its
53 voices cover English, Spanish, French, Italian, Japanese, Portuguese and
Chinese, and stop there. So this pack recommends Piper (`tr_TR-fettah`, ~21 MB),
which does speak Turkish, and Settings badges it accordingly.

Kokoro is still offered, not hidden. Pointed at Turkish text it will produce
confident, fluent, badly-accented output — a legitimate thing to want to hear,
and not our call to forbid. German is in the same position for the same reason.
