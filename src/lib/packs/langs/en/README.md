# English

**Variety:** neutral international English. Spelling leans British (`-ise`,
`colour`) but the tutor accepts American forms without correcting them.

**Maintained by:** core team (Official pack).

Only `pack.ts` is required; this folder has no long-form docs yet. Add one by
dropping a markdown file here — see [`../es/`](../es/) for the reference layout
and [CONTRIBUTING.md](../../../../../CONTRIBUTING.md) for the rules.
