# German (Deutsch)

**Variety:** standard German (Hochdeutsch), as written and spoken in Germany.
Austrian and Swiss usage is recognised but not taught.

**Maintained by:** core team (Official pack).

Only `pack.ts` is required; this folder has no long-form docs yet. Cases and
separable verbs are the obvious first chapters — see [`../es/`](../es/) for the
reference layout and [CONTRIBUTING.md](../../../../../CONTRIBUTING.md) for the rules.
