# Indonesian (Bahasa Indonesia)

The reference language pack for Standard Indonesian (Bahasa Indonesia).

| File | What it is |
| --- | --- |
| `pack.ts` | The pack literal — the app's contract. Short, structured, always loaded. |
| [`pronunciation.md`](./pronunciation.md) | Long-form sound notes for the learner. |
| [`grammar.md`](./grammar.md) | Grammar the three `grammar` bullets have no room for. |
| [`register.md`](./register.md) | `prompt: true` — also fed to the tutor on every turn. |

Docs are shown in **Settings → Language**, index (`README.md`) first, then by
filename. Prefix with `01-`, `02-` if a language needs a reading order.

## Scope of this pack

This pack teaches Standard Indonesian (Bahasa Indonesia) as used in education,
government, media, and formal communication throughout Indonesia. Informal
expressions may be introduced when appropriate, but the primary focus is
standard usage. Regional languages and dialects such as Javanese, Sundanese,
Balinese, or regional slang are outside the scope of this pack.
