# French (Français)

**Variety:** standard metropolitan French. Québécois and other varieties are
understood but not taught.

**Maintained by:** core team (Official pack).

Only `pack.ts` is required; this folder has no long-form docs yet. Liaison and
the tu/vous split are the obvious first chapters — see [`../es/`](../es/) for the
reference layout and [CONTRIBUTING.md](../../../../../CONTRIBUTING.md) for the rules.
